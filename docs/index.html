三种翻译模式，覆盖所有场景

无论你是阅读整篇外文文章、查询单个词汇，还是需要快速了解页面要点，AI 翻译助手都能胜任。

全页面翻译

一键将整个网页内容翻译为目标语言，保持原始排版不变。支持中英互译及 12 种语言，阅读外文网站如同阅读母语。

安装指南

四步开始使用

无需注册，无需付费，几分钟即可完成安装配置。

01

打开扩展管理页

在 Chrome 浏览器地址栏输入 chrome://extensions/ 并回车。

02

开启开发者模式

在扩展管理页面右上角，打开「开发者模式」开关。

03

下载并加载扩展程序

从 GitHub 下载项目源码并解压，然后点击「加载已解压的扩展程序」，选择解压后的文件夹即可完成安装。

04

配置 API 密钥

点击工具栏图标，在弹窗中填入你的 API Key、Base URL 和模型名称，保存后即可使用。

兼容所有 OpenAI 协议的 API 服务商，包括 DeepSeek、OpenAI、Moonshot 等。只需填入对应的 API 地址和密钥即可。
常见问题

你可能想知道的

这个插件是免费的吗？

插件本身完全免费开源。翻译功能需要调用 AI API，因此会产生 API 调用费用，费用取决于你选择的服务商和模型。推荐使用 DeepSeek 等性价比高的服务商。

支持哪些 AI 服务商？

兼容所有 OpenAI 协议的 API 服务商，包括但不限于 OpenAI、DeepSeek、Moonshot（月之暗面）、智谱 AI、通义千问等。只需在设置中填入对应的 Base URL 和 API Key。

我的 API 密钥安全吗？

API 密钥仅存储在你本地浏览器的 chrome.storage 中，不会上传到任何服务器。所有翻译请求直接从你的浏览器发送到 AI 服务商，不经过第三方中转。

支持哪些语言？

支持中文、英文、日文、韩文、法文、德文、西班牙文、俄文、阿拉伯文、意大利文、葡萄牙文等 12 种语言互译，也支持自动检测源语言。
